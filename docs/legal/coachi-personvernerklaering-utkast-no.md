# Personvernerklæring for Coachi

**Sist oppdatert:** 10. mars 2026

Denne personvernerklæringen beskriver hvordan Coachi behandler personopplysninger når du bruker tjenesten.

Coachi er en treningsapp som tilbyr løpecoaching, lydveiledning, treningshistorikk, konto, eventuelle Premium-funksjoner og support.

## 2. Hvilke opplysninger vi behandler
Vi kan behandle følgende opplysninger:

### Kontoopplysninger
- navn
- e-postadresse eller identifikator fra innloggingsleverandør
- intern bruker-ID
- innloggingsstatus og metode

### Profil- og treningsopplysninger
- fødselsdato eller alder
- språk
- nivå og brukerpreferanser
- treningsøkter, varighet, historikk og oppsummeringer
- pulsdata når tilgjengelig
- coach score og treningsrelaterte målinger

### Lyd- og interaksjonsdata
- lydopptak når du bruker stemmefunksjoner
- transkripsjon av tale
- svar generert av systemet
- teknisk informasjon om lydpakker og avspilling

### Abonnementsopplysninger
- abonnementstatus når betalte abonnement er aktive
- prøveperiode når dette tilbys
- kjøpsrelaterte metadata
- produktnivå

### Tekniske opplysninger
- appversjon
- enhetstype og operativsystem
- feil- og stabilitetslogger
- informasjon om synk og lydpakkeversjon

### Supportopplysninger
- meldinger du sender til support
- vedlegg og feilbeskrivelser du velger å dele

## 3. Formål med behandlingen
Vi behandler opplysninger for å:
- levere coaching og treningsfunksjoner
- lagre og vise treningshistorikk
- gi treningsoppsummeringer og innsikt
- administrere konto og innlogging
- håndtere abonnement og Premium-funksjoner når disse er aktive
- synkronisere lydpakker og appinnhold
- yte support
- forbedre stabilitet, sikkerhet og kvalitet
- forhindre misbruk

## 4. Behandlingsgrunnlag
Vi behandler personopplysninger på følgende grunnlag, avhengig av situasjonen:
- **Avtale**, når behandlingen er nødvendig for å levere tjenesten
- **Samtykke**, når dette kreves for eksempel for mikrofon, helse- eller sensorrelaterte tillatelser
- **Berettiget interesse**, for sikkerhet, feilretting og produktforbedring
- **Rettslig forpliktelse**, når loven krever oppbevaring eller utlevering

## 5. Helse- og treningsrelaterte data
Coachi kan bruke trenings- og pulsdata for å levere coaching og oppsummeringer. Slike data brukes bare i den grad det er nødvendig for appens funksjoner.

Hvis behandling av bestemte opplysninger krever særskilt samtykke, vil dette innhentes gjennom appen eller operativsystemet.

## 6. Lagringstid
Vi lagrer opplysninger så lenge det er nødvendig for formålet, eller så lenge loven krever det.

Generelt:
- kontoopplysninger lagres så lenge kontoen er aktiv
- treningshistorikk lagres så lenge tjenesten eller produktnivået tilsier det
- supporthenvendelser lagres så lenge det er nødvendig for saken
- tekniske logger lagres normalt kortere

Ved sletting av konto vil vi slette eller anonymisere opplysninger så langt det er mulig og lovlig.

## 7. Deling og databehandlere
Vi deler ikke personopplysninger med andre enn det som er nødvendig for å levere tjenesten, oppfylle lovkrav eller beskytte rettigheter.

Vi kan bruke databehandlere for:
- hosting og drift: Render
- lydlagring og innholdssynk: Cloudflare R2
- tekst-til-tale: ElevenLabs
- tale-til-tekst og AI-funksjoner: xAI, Google, OpenAI og Anthropic når disse er aktivert
- analyse og feilovervåking: PostHog og Sentry når disse er aktivert
- abonnement og appdistribusjon: Apple
- innlogging: Apple
- e-post og support: Resend eller konfigurert SMTP-leverandør når e-post er aktivert

Ved overføring utenfor EØS skal vi bruke gyldig overføringsgrunnlag.
