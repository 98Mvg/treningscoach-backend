# Coachi App Review Notes Template

Use this as the starting point for the App Review notes field in App Store Connect.

## What reviewers should know

- Coachi is free to download and has a free core workout experience.
- Premium is optional and offered as monthly or yearly Apple subscriptions.
- No paywall interrupts an active workout.

## Where to test subscriptions

1. Open the app and complete onboarding.
2. Navigate to profile/settings or the post-workout summary.
3. Open the Premium/paywall surface.
4. Use the visible `Restore Purchases` and `Manage Subscription` actions if needed.

## Where to test the free experience

1. Start a workout without purchasing Premium.
2. Complete a basic workout and open the summary.
3. Confirm the core coaching flow works without payment.

## Account actions

- Apple Sign-In and passwordless email are available for account-linked features.
- In-app account deletion is available from settings via `Delete account`.

## Notes

- If a reviewer wants to test Premium purchase and restore, monthly and yearly plans are in the same subscription group.
- If a reviewer wants to test the free version only, no purchase is required to start and complete the core workout flow.
