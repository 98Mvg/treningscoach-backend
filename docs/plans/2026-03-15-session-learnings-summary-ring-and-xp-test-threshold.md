# 2026-03-15 - Summary Ring Clarity And XP Test Threshold

- The workout summary score ring should show the final Coach Score directly. The full-sweep and count-up animation reads like a fake score roll rather than a completed result on this screen.
- XP feedback in the summary should stay inside the ring/level treatment instead of adding an extra bottom card when it does not carry essential information. Secondary XP cards quickly make the summary feel heavier than the score itself.
- Home and workout summary need different score-ring behavior even when they share the same ring component. Summary can disable count-up locally without creating a second ring architecture.
- For local gamification testing, the XP award duration threshold can be temporarily lowered in config as long as the threshold remains centralized in one place and covered by contracts.
