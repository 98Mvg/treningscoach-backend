Personvernerklæring for Coachi

1. Hvem vi er

Coachi er en treningsapp for løping og annen utholdenhetstrening. Tjenesten tilbyr onboarding, treningsoppsett, lydcoaching, økthistorikk, oppsummeringer, eventuelle Premium-funksjoner og support.

2. Hvilke opplysninger vi kan behandle

Konto og profil

Navn, e-postadresse, bruker-ID, språk, innloggingsstatus og profilopplysninger som alder, makspuls og hvilepuls.

Trening og historikk

Økter, varighet, coach score, pulsdata når tilgjengelig, oppsummeringer og treningshistorikk i appen.

Lyd og interaksjon

Lyd, transkripsjon og systemgenererte svar når du bruker tale- eller AI-funksjoner.

Teknisk drift

Appversjon, enhetstype, feillogger, synkstatus og supporthenvendelser du velger å sende inn.

3. Historikk og data i Coachi

Coachi lagrer treningshistorikk, oppsummeringer og relevante målinger for å vise fremgang over tid. Dette inkluderer blant annet varighet, pulsdata når tilgjengelig, coach score og andre felt som brukes i oppsummeringer og historikkvisninger i appen.

  • Historikken brukes for å vise deg utvikling, tidligere økter og mer relevante innsikter.
  • Noe som vises i appen kan avhenge av kontostatus, sensortilgang og produktnivå.
  • Du kan be om sletting eller innsyn i lagrede opplysninger ved å kontakte oss.

4. Hvorfor vi bruker opplysningene

  • For å levere coaching, struktur og oppsummeringer under og etter trening.
  • For å lagre historikk, vise fremgang og administrere konto og abonnement.
  • For å holde tjenesten stabil, forbedre kvalitet og forebygge misbruk.
  • For å yte support når du kontakter oss.

5. Puls, lyd og AI-funksjoner

Coachi kan bruke puls- og treningsdata for å levere bedre coaching. Når du bruker tale- eller AI-funksjoner, kan lyd og transkripsjoner behandles av aktiverte leverandører for å levere funksjonen du har startet.

  • Puls er frivillig, men kan gi mer presis trening.
  • Hvis puls eller sensor mangler, kan Coachi fortsette med andre coachingmoduser.
  • Tale- og AI-funksjoner brukes bare i den grad det er nødvendig for funksjonen i appen.

6. Behandlingsgrunnlag og databehandlere

Coachi behandler personopplysninger på grunnlag av avtale, samtykke, berettiget interesse eller rettslig forpliktelse, avhengig av funksjonen og typen opplysninger det gjelder.

  • Hosting og drift: Render
  • Lydlagring og innholdssynk: Cloudflare R2
  • Tale og AI-funksjoner: aktiverte leverandører som ElevenLabs, xAI, Google, OpenAI og Anthropic
  • Analyse og feilovervåking: PostHog og Sentry når disse er aktivert
  • E-post: Resend eller annen konfigurert e-postleverandør når e-post er aktivert
  • Distribusjon og abonnement: Apple

7. Lagringstid, sletting og rettigheter

Opplysninger lagres så lenge det er nødvendig for å levere tjenesten, følge opp support eller oppfylle lovkrav. Når en konto slettes, vil vi slette eller anonymisere opplysninger så langt det er mulig og lovlig.

  • Du kan be om innsyn, retting, sletting, begrensning eller dataportabilitet der loven gir grunnlag.
  • Du kan trekke tilbake samtykke der behandlingen bygger på samtykke.
  • Du kan kontakte oss på AI.Coachi@hotmail.com dersom du vil bruke rettighetene dine.

8. Sikkerhet og endringer

Vi bruker tilgangskontroll, begrenset tilgang og andre sikkerhetstiltak der det er relevant. Hvis tjenesten eller regelverket endres vesentlig, kan denne erklæringen bli oppdatert.
